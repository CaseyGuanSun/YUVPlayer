YUV Player -- YUV播放器
================================================

### 工程说明
    VS2010工程，参考Yuvviewer及YUV Player Deluxe编写的YUV播放器

### 功能
    播放各种格式的YUV文件。已知支持格式有：
    Y;
    YUV420(I420);YV12;
    YUV422(I422);YV16;
    YUV444;
    YUY2(YUYV);YVYU;UYVY;VYUY;
    NV12(YUV420SP);NV21;
    NV16(YUV422SP);NV61
